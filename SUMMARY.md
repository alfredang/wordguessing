# WordWhiz - A fun emoji word guessing game for kids

A single-page HTML game where players guess words from emoji clues. Built for kids aged 9-13 with a colorful, playful UI.

## Live Demo

🎮 **https://alfredang.github.io/wordguessing/**

## Quick Start

```bash
open index.html
```

No build step required — just open the HTML file in any browser.

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Web Audio API for sound effects
- CSS animations for visual feedback
- localStorage for high score persistence
